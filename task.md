## 기술 스택
- 런타임: Node.js + Express (로컬 서버)
- 프론트엔드: 순수 HTML5 Canvas + Vanilla JS (프레임워크/TS 없음, 인라인 CSS만)
- AI: **Claude CLI 서브프로세스(`claude -p --output-format stream-json`)** — 로그인된 구독 인증 사용, API 키 불필요(추가비용 없음). `.env`의 `AI_MODE=api`로 전환 시 `@anthropic-ai/sdk` 사용 가능.
- 실시간: WebSocket (`ws`)
- 영구 저장: 커스텀 에이전트 → `custom-agents.json`
